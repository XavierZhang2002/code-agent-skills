# code-agent-skills

Backup of my active Agent skills (`~/.claude/skills`).

## Skills

- **agent-reach** — Web search and 13+ platform integrations (Twitter, YouTube, GitHub, 小红书, etc.)
- **arxiv-research** — Search and summarize arXiv papers
- **mkdocs-visual-optimization** — MkDocs Material structural and visual optimization, including custom Mermaid theming
- **zotero-research** — Search and extract from local Zotero library

## Usage

Copy a skill folder into `~/.claude/skills/` to activate it.
